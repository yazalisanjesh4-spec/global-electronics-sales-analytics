# Global Electronics Sales Analytics – Business Insights

## Project Summary

This project analyzes sales data from a global electronics retailer using Microsoft Excel and Power BI. The objective is to transform raw transactional data into meaningful business insights through interactive dashboards and KPI reporting.

---

# Business Questions

The dashboard helps answer the following questions:

- What is the overall sales performance?
- How much profit has the business generated?
- Which countries contribute the highest revenue?
- Which product categories perform the best?
- Which products generate the highest sales?
- How do sales change over time?
- Which stores contribute the most revenue?
- What is the Average Order Value (AOV)?

---

# Analytical Approach

The project follows a complete analytics workflow:

1. Imported multiple CSV files.
2. Cleaned and transformed data using Power Query.
3. Built relationships between tables to create a relational data model.
4. Created DAX measures for KPI calculations.
5. Designed interactive dashboards in Microsoft Excel and Power BI.
6. Generated business insights through visual analysis.
