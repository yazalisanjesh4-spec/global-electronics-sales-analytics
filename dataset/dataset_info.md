# Dataset Information

## Dataset

Global Electronics Retailer Dataset

## Description

This dataset represents sales transactions from a global electronics retailer operating across multiple countries. It contains transactional, customer, product, store, and exchange rate information used for business intelligence and sales analytics.

## Dataset Files

- Customers.csv
- Products.csv
- Sales.csv
- Stores.csv
- Exchange_Rates.csv

## Data Model

The dataset follows a relational model where:

- Sales is the fact table.
- Customers, Products, Stores, and Exchange Rates are dimension tables.

Relationships were created in Power BI to build a proper analytical data model.

## Business Objective

Analyze sales performance and build interactive dashboards to answer business questions related to:

- Revenue
- Profit
- Customer behavior
- Product performance
- Geographic sales
- Monthly trends
- Store performance
